Database Infrastructure Design Project

Project: Designed a database for a hypothetical hospital

Focus: Database Infrastructure, SQL

Grade: Distinction

Project Document: Database Infrastructure Design Report

Objective:

Designed a functional database for a hypothetical private hospital, delivering a proof of concept report to support the feasibility of the planned data infrastructure. The project demonstrated how the database can facilitate data analysis to provide actionable insights for managerial staff.

Role:

  • Project Leader
  • Managed timelines
  • Facilitated bi-weekly scrum meetings
  • Resolved project roadblocks

Skills Developed:

  • Agile collaboration
  • Researching operation requirements
  • Designed Entity Relationship Diagrams
  • Developed relational schemas and materialised views using Oracle SQL developer
  • Designed and populated SQL databases
  • Conducted data analysis using Power BI graphs

Outcome:

Achieved a distinction final grade for successfully designing a functional database that effectively met the stakeholder's requirements.
